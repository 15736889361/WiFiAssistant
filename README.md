# WiFiAssistant
WiFi助手是一款可以扫描二维码连接WiFi的小工具软件.   
支持ZUI分享的WiFi二维码(ZUK Z1, Z2, Z2 Pro)和标准的WiFi二维码(小米等).扫描ZUI分享的WiFi二维码后,可以再次以标准二维码分享的形式分享出去.
